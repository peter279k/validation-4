Psecio\Validation

The Psecio\Validation library aims to be a simple, lightweight validation library that makes it simpler to integrate robust validation into your application. Simply put, it wants to lower the barrier between you and ensuring your user input is what it should be.

Quick Start

To get the Validation library installed and be ready for use, you can use this composer require command:

composer require psecio/validation

You can then use it immediately to evaluate incoming data from a simple PHP array based on a provided set of rules:

<?php
$validator = \Psecio\Validate\Validator::getInstance();
$data = [
    'first_name' => 'Chris'
    'email' => 'info@websec.io'
];
$rules = [
    'first_name' => 'required',
    'email' => 'required|email'
];

if ($validator->execute($rules, $data) === true) {
    echo 'Validation successful!';
}

?>
